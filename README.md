# Sales Trend Visualization Project

## Overview

This project analyzes sales data using Python and Data Analytics techniques. The project focuses on identifying sales trends, top-performing products, and category-wise sales distribution through various visualizations.

## Objective

To analyze sales performance, identify trends, and generate meaningful business insights using data visualization techniques.

## Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset

The dataset contains the following fields:

- Date
- Product
- Category
- Sales

A sample sales dataset was created and stored in CSV format for analysis.

## Methodology

1. Created the sales dataset.
2. Converted date values into datetime format.
3. Grouped sales data by month.
4. Generated visualizations.
5. Analyzed trends and business insights.

## Visualizations

- Monthly Sales Trend (Line Chart)
- Top Products by Sales (Bar Chart)
- Sales by Category (Pie Chart)
- Correlation Heatmap

## Results

- Laptop recorded the highest sales.
- Electronics category generated the maximum revenue.
- Monthly sales showed a positive growth trend.
- Visualizations helped understand sales performance effectively.

## Outcome

The project successfully demonstrated how data analytics and visualization techniques can be used to analyze sales data and support decision-making.

## Conclusion

Sales Trend Visualization provides valuable insights into business performance. The project highlights the importance of data visualization in identifying trends and improving business strategies.
